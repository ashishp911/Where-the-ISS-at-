Where is the ISS rn?

Latitude : °
Longitude : °
Altitute : km
Velocity : kmph


This project does the following things

  1. Fetching data from wheretheiss website using fetch and displaying the latitude, longitude, altitude and velocity of the International Space Station.
  2. Importing the leaflet JS library to display a map on the browser with an icon which indicates the ISS.
  3. Using the setInterval function to call the function at a desired time interval to follow the path of the ISS.
